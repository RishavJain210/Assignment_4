Aim : Use two classes “ad” and “sd”, “ad” will change the color to blue, and “sd” will change the background color of elements to red. Now, both classes together will change the font style to italics use them in the paragraph tag.

CLASS : ad

This is a PARAGRAPH element with class 'ad'.

This class changes the color to BLUE.

CLASS : sd

This is a PARAGRAPH element with class 'sd'.

This class changes the BACKGROUND color to RED.

CLASSES : ad & sd

This is a PARAGRAPH element with classes both 'ad' and 'sd'.

This will have the properties of class 'ad' as well as 'sd' as well as in italics.
